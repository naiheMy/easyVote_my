# 部署说明

## 一、安装依赖

​	在终端执行此命令安装所有依赖

```bash
pip install -r requirements.txt
```

## 二、启动项目

​	在对应文件夹内终端中输入此命令运行项目

```
python .\app.py
```

​	执行后会自动在云服务器端创建mysql数据库以及相关数据表，需提前准备数据库信息填写到config.py中

## 三、进入前端

​	双击文件夹中的index.html，即可进入前端页面~

## 四、其他

​	本项目依赖与云服务器存储数据，如遇数据库操作相关问题可以联系作者。
